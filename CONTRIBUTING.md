# Contributing to ForestWatch Togo

Thank you for your interest in improving this project! 
Since this repository holds the production code of an AI Model, please ensure you follow these steps:

1. **Bug Reports & Feature Requests**: Open an issue first to discuss the change.
2. **Data & Models**: Do not PR large `.joblib` or dataset files. Pointers to Hugging Face or Cloud Storage are preferred.
3. **Pull Requests**:
   - Ensure the code passes basic linting.
   - Keep inputs/outputs consistent with the `LandCoverPredictor` class.
   - Respect the CC-BY-NC-4.0 licensing model.
